# Dev Workflow

1. **Develop locally**:

```bash
   python app.py
   *# Visit http://localhost:5000*
```

1. **Commit changes**:

```bash
   git add .
   git commit -m "Added inventory system"
   git push
```

1. **Deploy to Pi**:

```bash
   *# Option A: Manual*
   ssh pi@pi3.local "cd adventure-game && git pull && sudo systemctl restart adventure-game"
   
   *# Option B: Ansible*
   ansible-playbook -i inventory.yml playbooks/update-game.yml
```
