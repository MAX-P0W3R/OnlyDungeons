# Git Workflow

```bash
# 1. Install Git
sudo apt install git -y

# 2. Clone your repo
cd ~
git clone https://github.com/yourusername/adventure-game.git
cd adventure-game

# 3. Install dependencies
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### `requirements.txt`:
```
Flask==3.0.0
gunicorn==21.2.0
```
